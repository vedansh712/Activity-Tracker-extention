# Icons Directory

This directory contains the extension icons in different sizes:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon32.png` - 32x32 pixels (extension management)
- `icon48.png` - 48x48 pixels (extension management)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Icon Requirements

- PNG format
- Square dimensions
- Transparent background recommended
- Simple, recognizable design

## Creating Icons

You can create custom icons using any image editing software. The current placeholders can be replaced with proper icons that represent time tracking or activity monitoring (e.g., stopwatch, clock, chart icons).

For now, you'll need to add actual PNG files with these names to make the extension fully functional.
